# Telegram Stars Bot

Простой бот на Python для покупки и продажи звёзд через встроенные платежи Telegram (Stars). Управление происходит через инлайн-кнопки: пользователь выбирает нужное действие, бот выставляет счёт, а успешный платёж сразу пополняет внутренний баланс.

## Возможности
- Кнопки для быстрой покупки фиксированных пакетов звёзд с оплатой внутри Telegram.
- Кнопки для продажи звёзд с проверкой остатка на балансе.
- Просмотр текущего баланса.
- Обработка pre-checkout и успешных платежей.

## Как запустить
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Укажите токен бота в переменной окружения `BOT_TOKEN` (создайте бота через [@BotFather](https://t.me/BotFather)).
3. Запустите бота:
   ```bash
   python bot.py
   ```

Для оплаты звёздами используется `provider_token="STARS"` и валюта `XTR`, поэтому все платежи проходят внутри Telegram.
